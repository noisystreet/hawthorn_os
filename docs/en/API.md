# Public API index

> **[中文](../API.md)** — Chinese source of this document.

**Hawthorn (山楂)** has not yet published a stable crate-level API. The following will be filled in as code lands.

| Area | Description | Status |
|------|---------------|--------|
| Syscall ABI | Register convention, syscall numbers, error codes | See [SYSCALL_ABI.md](./SYSCALL_ABI.md) |
| `syscall_abi` crate | User stubs and constants | Planned; see [ARCHITECTURE.md §8](./ARCHITECTURE.md) |
| Internal kernel crates | Module APIs under `kernel/` | Unstable; kernel build only |
